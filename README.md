# PickMyFate 🔮

A mystical decision-making tool that chooses your fate when you can't decide for yourself.

## What It Does

Enter two or more options — one per line — and let PickMyFate dramatically reveal your destiny. Features a shuffling animation, ambient particle effects, and cryptic verdicts to make even the smallest decision feel momentous.

## How to Use

1. Type your options into the scroll, one per line
2. Toggle **BEST OF 3** on if you want three dramatic rounds instead of one
3. Click **CONSULT THE ORACLE** (or press `Cmd + Enter`)
4. Watch fate decide for you
5. Not satisfied? Click **ASK AGAIN**

## Features

- 🎲 Random selection with dramatic shuffle animation
- 🏆 Best of 3 mode — three rounds, each shuffling for a different length of time
- ✨ Ambient gold particle effects
- 🔮 Cryptic Oracle verdicts
- 📱 Works on desktop and mobile

## Run Locally

No dependencies, no build step. Just open the file:

```bash
open index.html
```

Or drag `index.html` into any browser.

## Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://MikeyBlueEyes123.github.io/pickmyfate`
